- Deskripsi Repository: Repository ini digunakan sebagai Tugas 3 Git Collaboration Practice atau Praktik Kolaborasi GitHub pada mata kuliah Software Development. Setiap anggota kelompok melakukan clone repository kelompok ke komputer masing-masing menggunakan Visual Studio Code, membuat branch masing-masing, membuat file pada branch, melakukan commit perubahan pada branch masing-masing, melakukan push branch ke GitHub, membuat pull request dari branch masing-masing menuju branch main, dan merge pull request untuk memahami proses workflow kolaborasi Git dalam proyek pengembangan perangkat lunak.
